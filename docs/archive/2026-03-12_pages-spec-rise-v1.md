# Salesboard v2 — 페이지별 기능 명세 (RISE 설계안)

> 역할별 접근 가능 페이지가 다름. 공통 → 역할별 순서로 정리.

---

## 🌐 공통

### 1. 로그인 / 회원가입
- Supabase Auth (이메일+비밀번호, Magic Link)
- 역할별 자동 라우팅 (로그인 후 각자 대시보드로)
- 바이어 회원가입: 관리자 승인 후 활성화
- 다국어 지원 (i18n — 영어 기본, 추후 확장)

### 2. 제품 카탈로그 `/catalog`
- 전체 제품 그리드/리스트 뷰 전환
- 브랜드별, 카테고리별 필터
- 제품 카드: 썸네일, 품목명, 용량, 브랜드
- 검색 (품목명, SKU, 바코드)

### 3. 제품 상세 `/catalog/:id`
- 제품 이미지 갤러리 (product_media)
- 기본 정보: SKU, 바코드, 용량, 브랜드
- 국가별 탭: 성분표기, 사용법, 주의사항 (product_locales)
- 물류 규격: CBM, 중량, 입수량, 박스 사이즈, HS코드
- 컨텐츠 영역: 제품 소개, 마케팅 자료, 영상
- **바이어**: "장바구니 추가" 버튼
- **영업/관리자**: 제품 정보 편집 가능

### 4. 내 프로필 `/profile`
- 기본 정보 수정
- 비밀번호 변경
- 알림 설정

## 🛒 바이어 (Buyer)

### 5. 바이어 대시보드 `/buyer`
- 최근 주문 요약 (최근 5건, 상태별 색상)
- 진행 중 주문 현황 (submitted, negotiation 등)
- 출하 트래킹 (배송 중인 건)
- 담당 영업사원 정보 + 빠른 문의 버튼
- 공지사항 / 신제품 알림

### 6. 발주하기 `/buyer/order/new`
- **장바구니 컨셉**: 카탈로그에서 추가한 품목 목록
- 품목별 수량 입력 (입수량 단위 가이드 표시)
- 희망 납기일 선택
- 출고지(지사) 선택 (organizations branch)
- 메모 입력
- "발주 요청" → status: `submitted`
- **재주문**: 과거 주문에서 "동일 발주" 원클릭

### 7. 내 주문 목록 `/buyer/orders`
- 전체 주문 이력 (필터: 기간, 상태)
- 상태 배지: draft, submitted, negotiation, confirmed, shipped...
- 주문번호 클릭 → 상세

### 8. 주문 상세 `/buyer/orders/:id`
- 주문 품목 리스트 (희망수량 vs 확정수량 비교)
- 상태 타임라인 (order_events 기반)
- **네고 구간**: 영업이 수량/납기/유통기한 조정 제안 → 바이어 수락/거절
  - "유통기한 2026-09-30 로트 괜찮으시겠습니까?" → 수락/거절 UI
  - 수량 조정 제안 → 확인 UI
- 인보이스 다운로드 (발행 후)
- 출하 정보 + 트래킹
- 서류 다운로드 (패킹리스트, 원산지증명 등)

### 9. 바이어 실적 대시보드 `/buyer/analytics`
- 월별/분기별 구매 실적 차트
- 품목별 구매 비중 (파이차트)
- 연간 누적 구매액
- 전년 대비 성장률

### 10. 문의하기 `/buyer/inquiry`
- 문의 작성 (주문 연결 선택 가능)
- 문의 이력 조회
- 답변 확인

### 11. 미팅 예약 `/buyer/meeting` *(MVP 이후)*
- 캘린더에서 가용 시간 선택
- 미팅 목적, 필요 통역 언어 선택
- 예약 상태 확인

## 🤝 벤더 (Vendor)

### 12. 벤더 대시보드 `/vendor`
- 담당 바이어 목록 + 각 바이어 최근 주문 상태
- 월간 매출 요약 (담당 바이어 합산)
- 커미션 현황 (이번 달 예상 / 확정 / 지급완료)
- 목표 매출 vs 실적 게이지

### 13. 바이어 주문 관리 `/vendor/orders`
- 담당 바이어들의 발주 목록
- status: `submitted` → 벤더 컨펌 필요한 건 하이라이트
- 컨펌 / 반려 액션
- 컨펌 → status: `vendor_review` → `sales_review`로 전달

### 14. 매출 모니터링 `/vendor/analytics`
- 바이어별 매출 추이
- 품목별 판매 현황
- 기간별 필터 (월/분기/연)
- 목표 대비 달성률

### 15. 커미션 관리 `/vendor/commissions`
- 주문별 커미션 내역
- 상태별 필터 (pending, confirmed, paid)
- 월별 커미션 합산
- 커미션 명세서 다운로드

### 16. 제품 정보 열람 `/vendor/catalog`
- 카탈로그 동일 (편집 불가, 열람만)
- 제품별 마케팅 자료 다운로드
- 바이어에게 공유할 자료 선택

## 📊 영업담당자 (Sales)

### 17. 영업 대시보드 `/sales`
- **오늘의 할일**: 컨펌 대기 발주, 미답변 문의, 예정 미팅
- 전체 파이프라인 요약 (단계별 발주 건수/금액)
- 담당 바이어별 최근 활동
- 월간 실적 KPI (목표 vs 실적)

### 18. 발주 검토 `/sales/orders`
- 전체 담당 발주 목록 (필터: 상태, 바이어, 기간)
- **핵심 뷰**: 발주 건별로 아래 정보 한눈에
  - 바이어 희망 품목/수량
  - 출고 가능 재고 (lot별 유통기한 포함)
  - 생산 예정 일정
  - 재고 부족 품목 경고 🔴
  - 유통기한 보더라인 품목 경고 🟡

### 19. 발주 검토 상세 `/sales/orders/:id`
- 품목별 검토 패널:
  - 희망수량 vs 가용재고 vs 생산예정 비교 테이블
  - lot 선택 (유통기한 확인, 자동 추천 + 수동 변경)
  - 수량 조정 (사유 입력 필수)
  - 납기일 조정
- **액션 버튼**: 
  - "전체 컨펌" → confirmed
  - "부분 컨펌" → 아이템별 confirmed/rejected/partial
  - "바이어 협의 요청" → negotiation (메시지 전송)
- 컨펌 → 인보이스 발행 버튼 활성화

### 20. 인보이스 관리 `/sales/invoices`
- 인보이스 생성 (확정된 주문 → 자동 품목/금액 매핑)
- 결제 조건 설정 (Net 30, T/T 등)
- PDF 생성 + 다운로드
- 발송 (이메일)
- 결제 상태 추적 (pending → paid)

### 21. CRM / 바이어 관리 `/sales/crm`
- 담당 바이어 전체 리스트
- 바이어 카드: 최근 주문, 총 거래액, 마지막 활동일
- 바이어별 상세: 주문 이력, 문의 이력, 미팅 이력
- **AI 인사이트 패널** *(Agentic Sales)*:
  - 구매 패턴 분석 ("3개월째 Toner 미발주, 재고 소진 예상")
  - 신제품 추천 (바이어 구매 이력 기반)
  - 레터 초안 자동 생성
  - 재발주 타이밍 알림

### 22. 영업 실적 `/sales/analytics`
- 담당 바이어별 매출 차트
- 품목별 판매 순위
- 월별/분기별 실적 추이
- 목표 달성률
- 벤더별 커미션 요약

### 23. 문의 관리 `/sales/inquiries`
- 담당 바이어 문의 리스트
- 미답변 건 하이라이트
- 답변 작성 (AI 초안 제안)
- 상태 관리 (open → in_progress → resolved)

## 📦 물류담당자 (Logistics)

### 24. 물류 대시보드 `/logistics`
- 확정 주문 중 출하 준비 필요한 건 목록
- 진행 중 출하 현황
- 오늘/이번 주 출하 예정

### 25. 출하 관리 `/logistics/shipments`
- 확정된 주문 → 출하 건 생성
- 출하 건별: 포워더, 선적 방법(FCL/LCL/Air), ETD, ETA
- 출고지(지사)별 분류 뷰
- 트래킹 번호 입력

### 26. 팔레트 적재 `/logistics/shipments/:id/packing`
- **핵심 기능**: 카툰 사이즈 → 팔레트 적재 시뮬레이션
  - 제품별 case_size 자동 로드
  - 팔레트 규격 선택 (1100×1100, 1200×1000 등)
  - 적재 시뮬레이션 뷰 (2D/3D)
  - CBM, 중량 자동 계산
- 팔레트 생성 + 품목 배치
- **lot 매칭**:
  - 재고 lot → pallet_items 자동 매칭 (FIFO 기본)
  - 유통기한 자동 기재
  - ⚠️ 전산재고 불확실 시 수기 조정 가능
- **쉬핑마크 자동 발급**:
  - 팔레트별 쉬핑마크 생성 (바이어 정보 + 유통기한 + 로트)
  - PDF 출력

### 27. 패킹리스트 `/logistics/shipments/:id/packing-list`
- 팔레트별 품목/수량/lot/유통기한 자동 집계
- 출고지별 분류
- PDF 생성 + 다운로드
- 바이어/포워더 공유

### 28. 서류 관리 `/logistics/shipments/:id/documents`
- 원산지 증명서 발급/업로드
- B/L 업로드
- 통관 서류
- 포워더 발급 서류 보관
- 트래킹 정보 업데이트

## ⚙️ 총괄 관리자 (Admin)

### 29. 관리자 대시보드 `/admin`
- 전사 주문 현황 (파이프라인 전체)
- 매출 요약 (일/주/월)
- 재고 현황 요약 (부족 품목 알림)
- 사용자 활동 로그

### 30. 사용자 관리 `/admin/users`
- 전체 사용자 CRUD
- 역할 배정/변경
- 바이어 가입 승인
- 계정 활성화/비활성화
- 벤더-바이어-영업 매핑 관리

### 31. 조직 관리 `/admin/organizations`
- 국가 → 업체 → 지사 트리 뷰
- 조직 CRUD
- 조직별 사용자 배정

### 32. 제품 관리 `/admin/products`
- 제품 CRUD
- 국가별 정보 관리
- 미디어/컨텐츠 업로드
- 가격 정책 관리 (바이어/벤더별 단가 설정)

### 33. 재고 현황 `/admin/inventory`
- 전체 재고 현황 (MES 동기화 상태 표시)
- lot별 유통기한 조회
- 부족 재고 알림
- 생산 예정 현황
- 동기화 수동 트리거

### 34. 전체 주문 관리 `/admin/orders`
- 모든 주문 조회 (필터: 바이어, 영업, 상태, 기간)
- 주문 상태 강제 변경 (비상시)
- 통계/리포트 생성

### 35. 정산 관리 `/admin/finance`
- 인보이스 전체 목록
- 결제 현황 추적
- 미수금 관리
- 커미션 확정/지급 처리
- 정산 리포트

### 36. 시스템 설정 `/admin/settings`
- 주문번호 채번 규칙
- 이메일 알림 설정
- MES 동기화 설정 (주기, 필터)
- 다국어 설정
- 환율 관리
